# Docking — what we'd like to use (plain-language brief for the instructors)

## How our docking idea works (in one breath)
The docking station has a **small light** on it. Our robot already has **two light sensors**, so it steers toward that light from across the room. When it gets close, a **funnel-shaped guide** on the station nudges it into the exact spot. **Two magnets** — one on the station, one on the robot — then snap together so it parks perfectly and holds still. A **small magnet-detector** on the robot notices the magnets have joined, the robot checks it has fully stopped, and then it **flashes a light or beeps** to announce "docked." We'll make the actual target the **smallest allowed (1 cm × 1 cm)**, so simply parking on the magnet covers it completely.

## What we'd need — core items
1. **A small bright light** for the docking station (the thing the robot drives toward). Ideally one we can make **blink**, so the robot can tell it apart from the room lighting.
2. **Two magnets** — one to attach to the robot, one for the docking station.
3. **A small "magnet-detector"** for the robot — so it knows the exact moment it has docked.
4. **Stiff material to build a funnel-shaped guide** on the station (cardboard, foam, or plastic) and some **colored tape**.
5. **A way for the robot to signal it's finished** — a small light or a beeper.

> Already on the robot (no need to request): the **two light sensors** and the **wheels/motors**. We may just need to position the light sensors so they face forward.

## Nice-to-have / backups (only if easy to get)
- **A "motion sensor"** to double-check the robot has fully stopped before it signals (so we don't lose points for signalling while still moving).
- **A colour sensor + colored tape**, as an alternative guiding method if lights or magnets aren't allowed.
- **A camera**, as a more complex backup option.

## Permissions we need to confirm with you
1. May we **attach things to the robot** (a magnet, the magnet-detector, and reposition its light sensors)?
2. May we **build the docking station with a physical funnel** and put **tape on the floor** around it?
3. Is it OK for the docking station to have a **small powered light** on it?
4. Is there a **size limit** on the docking station / funnel?
